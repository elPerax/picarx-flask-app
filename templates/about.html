About this project

This IoT project uses a Raspberry Pi and the PiCarX robot car. The Pi sends sensor data (ultrasonic distance, grayscale sensors, and TTS text) to Adafruit IO and also stores it in a local SQLite database. When Internet is available, the Pi synchronizes the local data with a Neon PostgreSQL database in the cloud.

This Flask web app connects to:

  • Adafruit IO (for live sensor values and remote control).
  • Neon (for querying and plotting historical sensor data).
