About VisionTrack-X

VisionTrack-X is an IoT and robotics project built on the PiCarX robot platform and a Raspberry Pi. The rover is capable of real-time sensing, obstacle avoidance, line following, and live telemetry streaming to the cloud.

The car continuously publishes ultrasonic distance, grayscale line-tracking values, and text-to-speech messages to Adafruit IO. It stores data locally in SQLite and synchronizes it to a managed Neon PostgreSQL database whenever Internet connectivity is available.

This Flask dashboard provides real-time monitoring, historical data analysis, and remote control for motors, steering, camera, and TTS.

Hardware

  • SunFounder PiCar-X robot chassis
  • Raspberry Pi with Robot HAT
  • Ultrasonic distance sensor
  • Grayscale line-tracking sensor array
  • Pan-tilt camera + speaker

Software & Cloud

  • Python robot control scripts
  • Flask dashboard
  • Adafruit IO (MQTT) for live telemetry
  • Neon PostgreSQL for cloud storage
  • Render for cloud deployment
